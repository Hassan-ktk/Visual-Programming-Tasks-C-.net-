# Visual-Programming-Tasks-C-Sharp-Asp.net-
This repository contains Visual Programming tasks including Windows Forms and ASP.NET Web Forms CRUD, DBCon usage, Three-Tier Architecture, Stored Procedures, LINQ to List CRUD, ASP.NET Core MVC with EF Code-First, RESTful APIs using Repository Pattern, API consumption with LINQ to JSON, and Authentication &amp; Authorization (Connection-less,Oriented)

**Note:**

This project ZIP file was split into multiple parts because of GitHub’s file size limit.
Requirements: 1.Bash or Any Terminal that support cat command.
Step1: Download the spit parts. Make sure the file name are same. Example: project_part_aa,project_part_ab.
step2: Run this command : cat project_part_* > project.zip
step3: Extract the Project. Command :unzip project.zip


**Project Details:**
**1.Simple Windows Forms CRUD Application**
A Windows Forms application developed in C#
Performs Create, Read, Update, and Delete (CRUD) operations
Uses ADO.NET for database connectivity

**2.ASP.NET Web Forms CRUD Application**
A web-based application using ASP.NET Web Forms
Implements basic CRUD operations
Uses ADO.NET for database access

**3.ASP.NET Web Forms CRUD using DBCon File**
Web Forms application with a separate DBCon (Database Connection) file
Implements CRUD operations
Demonstrates better code reusability and separation of concerns

**4.Three-Tier Architecture Application**
Implements Three-Tier Architecture
AppProps (User) – Entity / Model
Data Access Layer (DBCon, UserDAL)
Business Logic Layer (UserBLL)
Presentation Layer
Presentation layer implemented in both Windows Forms and Web Forms

**5.CRUD Operations using Stored Procedures (Master-Page)**
CRUD operations implemented using SQL Stored Procedures
Application developed using ADO.NET and ASP.NET Web Forms using Master Page
Focuses on performance, security, and structured database access

**6.LINQ to List CRUD Application**
Application developed using LINQ to List
Performs CRUD operations on in-memory collections
Demonstrates lambda expressions and LINQ queries

**7.ASP.NET Core MVC with Entity Framework (Code First)**
Web application developed using ASP.NET Core MVC
Uses Entity Framework Core (Code First Approach)
Performs full CRUD operations

**8.RESTful APIs using ASP.NET Core**
Designed RESTful Web APIs
Uses ASP.NET Core and Entity Framework
Implements Repository Design Pattern
Provides clean separation between API logic and data access

**9.Consuming REST APIs in ASP.NET Core MVC**
ASP.NET Core MVC application to consume APIs
Uses LINQ to JSON with Newtonsoft.Json package
Demonstrates API integration and JSON handling

**10.Authentication and Authorization Applications**
Two separate implementations:
a. Connection-Oriented Authentication (ADO.NET)
b. Connection-Less Authentication (Disconnected Architecture)
Developed using ASP.NET Web Forms
Implements login, session handling, and authorization logic
